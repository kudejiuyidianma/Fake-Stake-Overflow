# how to run

1. delete old fake_so mongodb database
2. `cd server`
3. `npm install`
4. `node populate_db.js mongodb://127.0.0.1:27017/fake_so`
5. `node server.js`
6. open another terminal, `cd client`
7. `npm install`
8. `npm start`
